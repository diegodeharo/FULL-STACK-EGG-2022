PAQUETE-> nos ayudan a organizar los conjuntos de CLASES y nos asisten para poder compartir el codigo.
CLASE-> la clase java es la forma esencial en la cual el lenguaje nos permite organizar nuestro codigo;
	 dentro de las Clases se coloca los datos y codigo del programa que vamos a codificar
METODO MAIN-> nos permite ejecutar nuestro programa
MAIN-> es el primer subprograma que se llama cuando invoco un programa java
METODO-> es igual a una funcion o un procedimiento, solo que es la manera de nombrar a los subprogramas que 
	se CODIFICAN dentro de una clase.
